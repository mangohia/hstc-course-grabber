# 🎯 韩师抢课助手

**韩山师范学院选修课自动抢课脚本** — 输入课程名 → 设置时间 → 自动抢课。支持自动翻页、页面保活刷新、抢完可继续抢其他课。

## 📥 安装方法（详细步骤）

### 🔧 第一步：安装 Tampermonkey（篡改猴）

**这是什么？** Tampermonkey 是一个浏览器扩展，它让我们写的抢课脚本能在网页上自动运行。

**安装步骤：**

| 你的浏览器 | 操作 |
|:----------|:-----|
| **Chrome** (谷歌浏览器) | 打开这个网址 → 点「添加至 Chrome」→「添加扩展程序」<br>https://www.tampermonkey.net/index.php?browser=chrome |
| **Edge** (微软浏览器) | 打开这个网址 → 点「获取」→「添加扩展」<br>https://www.tampermonkey.net/index.php?browser=edge |
| **360浏览器 / QQ浏览器** | 打开上面的 Chrome 链接，一样的操作 |

**安装成功标志：** 浏览器右上角出现一个 🐒 图标（可能有数字角标）

> ⚠️ **安装后必做！开放权限：**
> ① 点浏览器右上角 🧩 **拼图图标** → **管理扩展程序**
> ② 找到 **Tampermonkey** → 点 **「详情」**
> ③ 往下翻 → 打开 **「允许访问文件网址」** 开关
> ④ 回到选课页面刷新一下
>
> ❌ 如果不打开这个开关，脚本会没反应！

---

### 📜 第二步：安装抢课脚本

**方法一：一键安装（推荐）**

👉 **[点击此处安装韩师抢课助手](https://gitee.com/mangohia/hstc-course-grabber/raw/main/%E9%9F%A9%E5%B8%88%E6%8A%A2%E8%AF%BE%E5%8A%A9%E6%89%8B.user.js)**

点击后会出现 Tampermonkey 的安装页面，点 **「安装」** 即可。

**方法二：手动安装（如果一键安装没反应）**

1. 右键点 [安装链接](https://gitee.com/mangohia/hstc-course-grabber/raw/main/%E9%9F%A9%E5%B8%88%E6%8A%A2%E8%AF%BE%E5%8A%A9%E6%89%8B.user.js) → 「另存为」→ 保存到桌面
2. 点浏览器右上角 🐒 图标 → **「管理面板」（Dashboard）**
3. 点 **「➕ 添加新脚本」** 标签
4. 删掉编辑器里默认出现的代码
5. 用记事本打开刚才下载的文件 → **全选复制（Ctrl+A → Ctrl+C）**
6. 回到浏览器 → **粘贴（Ctrl+V）**
7. **Ctrl+S** 保存

**安装成功标志：** 🐒 图标里能看到「韩师抢课助手」的选项

---

> ⚠️ **如果安装后不生效：**
> 浏览器右上角拼图图标 → 管理扩展程序 → Tampermonkey → 详情 → 打开 **「允许访问文件网址」**

## 🚀 使用方法

### 基础用法（手动抢课）

```
┌─────────────────────────────────┐
│ 流程概览：                       │
│                                 │
│  ① 登录教务系统                  │
│  ② 进入选课页面                  │
│  ③ 输入要抢的课程名              │
│  ④ 点「⚡ 立即开抢」             │
│  ⑤ 等着，抢到了会自动停          │
│  ⑥ 点「🔄 继续抢课」抢下一门     │
└─────────────────────────────────┘
```

**详细步骤：**

**① 登录** 韩山师范学院教务系统
```
https://webvpn.hstc.edu.cn
```
→ 输入学号密码 → 登录

**② 进入选课页面**
登录后 → 找到 **「学生选课」**或 **「网上选课」** → 点进去
> 页面 URL 里应该包含 `eams/stdElectCourse` 字样

**③ 等右上角出现蓝色面板**
```
┌─ 🎯 韩师抢课助手 v3 ──────────┐
│                                 │
│  输入课程名，选择开抢时间 🚀      │
│                                 │
│  ⏰ 开抢时间                     │
│  [14时▾] [00分▾]                │
│  ☑ 自动刷新页面保活 [30秒▾]      │
│                                 │
│  目标课程（每行一个）：           │
│  ┌─────────────────────────┐    │
│  │ 舌尖上的潮州菜            │    │
│  │ 财经新闻与理财             │    │
│  └─────────────────────────┘    │
│                                 │
│  [ 🚀 启动自动模式 ]             │
│          或                      │
│  [ ⚡ 立即开抢（手动）]          │
└─────────────────────────────────┘
```

**④ 输入课程名**
- 每行一个课程名
- 课程名不需要完全一样，脚本会自动在当前页和后面几页找
- 如果知道全名最好，不确定的话输入关键词（比如"潮州菜"）

**⑤ 选择抢课方式**

| 按钮 | 什么时候用 | 效果 |
|:----|:----------|:-----|
| **🚀 启动自动模式** | 想设好时间让它到点自动抢 | 设置时间 → 自动刷新页面保活 → 到点开抢 |
| **⚡ 立即开抢（手动）** | 马上开抢，不等倒计时 | 立刻扫描表格找课 → 找到就点选课 |

**⑥ 看结果**
- 每个课程后面会显示：⏳等待中 → 🔍未出现/🎯找到 → ✅已抢到
- 全部抢完后按钮变成 **「🔄 重新开抢」**
- 点「重新开抢」可以输入新课名继续抢

---

### 自动模式（设好时间到点自动抢）

适合想提前设置好，到点自动开抢的场景：

```
① 输入课程名
② 设开抢时间（如14:00）
③ 勾选「自动刷新页面保活」
④ 点「🚀 启动自动模式」
⑤ 面板显示倒计时 ⏰ 00:32:15
⑥ 页面每30秒自动刷新一次
⑦ 到14:00:00 → 自动开抢！
```

> ⚠️ 页面不能关！自动刷新只是让页面不掉线，不是跑在后台。

---

### 抢完继续抢

```
① 第一轮：输入「舌尖上的潮州菜」→ 点开抢 → ✅ 抢到
② 按钮变成「🔄 继续抢课」
③ 点「继续抢课」→ 输入框回来
④ 输入「财经新闻与理财」→ 点开抢
⑤ ✅ 又抢到！
```

## ✨ 全部功能特性

| 功能 | 说明 |
|:----|:-----|
| ✅ **自由输入课程名** | 每行一个，想抢什么输什么 |
| ✅ **自动翻页** | 当前页没找到，自动翻下一页继续找 |
| ✅ **自动点击选课** | 找到课程后自动点「选课」 |
| ✅ **每秒重试** | 每0.5秒扫一次，持续10分钟 |
| ✅ **自动处理确认弹窗** | 弹出「确定」框后自动点确定 |
| ✅ **面板可拖动** | 按住标题栏拖到任意位置 |
| ✅ **随时停止** | 点「⏹ 停止」立刻终止抢课 |
| ✅ **停止后可重试** | 修改课程名，点一下就能继续抢 |
| ✅ **抢完继续抢** | 不用刷新，继续输入新课名 |
| ✅ **自动模式** | 设好时间、自动刷新页面保活、到点开抢 |
| ✅ **抢完自动确认** | 切换到「已选课程」标签查看结果 |
| ✅ **适配多浏览器** | Chrome / Edge / 360浏览器都能用 |

## 🔧 常见问题

### ❓ 面板没弹出来？
```
① 检查 🐒 图标是否开启（灰色→点一下变彩色）
② 检查 URL —— 必须是选课页面（含 eams/stdElectCourse）
③ 刷新页面试试
④ 重装脚本试试
```

### ❓ 显示「未出现」？
```
→ 课程还没显示在选课列表里（教务系统还没开放）
→ 脚本会自动翻页查找，等待即可
→ 如果是自动模式，到点后会自动刷新
```

### ❓ 页面一直刷新停不下来？
```
→ 点「✖ 取消自动刷新」按钮
→ 或者关闭页面重新进入
```

### ❓ 抢完了怎么抢别的课？
```
→ 点「🔄 继续抢课」
→ 输入框出现 → 输入新课名 → 点开抢
```

## 📝 使用技巧

- **提前10分钟登录**教务系统，进入选课页面
- **提前把课程名存备忘录**，抢课时直接复制粘贴
- **用自动模式**：设好14:00，勾选自动刷新，到点自动开抢
- **如果是手动抢**：时间一到就点「⚡ 立即开抢」
- **多门课分批次抢**：先抢最想要的，抢到后再抢备选的
