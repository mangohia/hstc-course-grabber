# 🎯 韩师抢课助手

**韩山师范学院选修课自动抢课脚本** — 输入课程名 → 等着 → 自动抢到。支持自动翻页、自动刷新保活、抢完可继续抢。

---

## ✨ 能干什么

| 功能 | 说明 |
|:----|:-----|
| ✅ **输入课程名或课程序号** | 每行一个，按课程名或序号都能找到 |
| ✅ **自动翻遍所有页** | 当前页没找到，自动翻下一页，一页不落 |
| ✅ **自动点击「选课」** | 找到课程后自动点，不用你动手 |
| ✅ **自动点确认弹窗** | 弹出「确定」框后自动点掉 |
| ✅ **0.5秒扫一次** | 速度很快，持续10分钟不放弃 |
| ✅ **及时停止** | 点「⏹ 停止」立刻停，修改课程可重来 |
| ✅ **抢完继续抢** | 不用刷新页面，继续输入新课名 |
| ✅ **自动模式** | 设好时间（如14:00），自动刷新保活，到点自动开抢 |
| ✅ **面板可拖动** | 按住标题栏拖到不挡视线的地方 |
| ✅ **多浏览器** | Chrome / Edge / 360都能用 |

---

## 🚀 快速开始（30秒学会）

```
装好脚本后 → 打开教务系统选课页面
→ 右上角出现蓝色面板
→ 输入课程名 → 点「⚡ 立即开抢」
→ 等着，抢到了会自动停
→ 点「🔄 继续抢课」抢下一门
```

## 📝 使用技巧

**⏰ 提前登录**
提前 10 分钟打开教务系统并登录，别卡在登录那步

**📋 提前存好课程名**
把要抢的课程名存在备忘录里（比如微信收藏），抢课时直接复制粘贴
```
舌尖上的潮州菜
财经新闻与理财
```

**🎯 先抢最想要的**
第一轮先抢最想上的课 → 抢到后点「继续抢课」→ 再抢备选的

**🤖 用自动模式省心**
如果还没到选课时间（比如14:00开抢）：
```
设时间14:00 → 勾选自动刷新 → 点「🚀 启动自动模式」
→ 页面会自己刷新保活 → 到14:00自动开抢
→ 你到点打开浏览器看一眼就行
```

**💡 课程名不用打全，也能输课程序号**
输入关键词就行，比如「潮州菜」也能匹配到「舌尖上的潮州菜」
如果知道课程序号（如 001），直接输入序号也能找到

## 📥 安装方法

### 1️⃣ 装 Tampermonkey（🐒 猴子插件）

Tampermonkey 是一个浏览器扩展，让脚本能在网页上自动运行。

| 浏览器 | 安装地址 |
|:-------|:---------|
| **Chrome** | https://www.tampermonkey.net/index.php?browser=chrome |
| **Edge** | https://www.tampermonkey.net/index.php?browser=edge |
| **360/QQ浏览器** | 跟 Chrome 一样，用上面的链接 |

点链接 → 点「Download」或「添加至 Chrome」→ 添加扩展程序

> **安装成功：** 浏览器右上角出现 🐒 图标

> ⚠️ **安装后必做这一步！**
> ① 点右上角 🧩 **拼图图标** → **管理扩展程序**
> ② 找到 **Tampermonkey** → 点 **「详情」**
> ③ 往下翻 → 打开 **「允许访问文件网址」** 开关
>
> ❌ 不打开这个开关，脚本不会运行！

### 2️⃣ 装抢课脚本

👉 **[点此一键安装](https://gitee.com/mangohia/hstc-course-grabber/raw/main/%E9%9F%A9%E5%B8%88%E6%8A%A2%E8%AF%BE%E5%8A%A9%E6%89%8B.user.js)**

点上面的链接 → 浏览器会弹出安装页面 → 点 **「安装」** → 完成！

> **手动安装（一键安装没反应时）：**
> ① 右键上面的链接 → 「另存为」→ 保存到桌面
> ② 点 🐒 图标 → **「管理面板」**
> ③ 点 **「➕ 添加新脚本」**
> ④ 删掉默认代码
> ⑤ 用记事本打开刚才下载的文件 → 全选复制（Ctrl+A → Ctrl+C）
> ⑥ 粘贴到浏览器（Ctrl+V）
> ⑦ **Ctrl+S** 保存

### 3️⃣ 开抢！

打开教务系统：https://webvpn.hstc.edu.cn
→ 登录 → 进入选课页面（URL含 `eams/stdElectCourse`）
→ 右上角出现蓝色面板 → 输入课程名 → 开抢！🎯

## ❓ 常见问题

**面板没弹出来？**
```
① 检查 🐒 是否灰色 → 点一下变彩色
② 检查 URL 是否为选课页面（含 eams）
③ 检查「允许访问文件网址」是否打开
④ 刷新页面
⑤ 重装脚本
```

**显示「未出现」？**
→ 课程还没显示，脚本会自动翻页查找，等一等
→ 如果是自动模式，到点后页面会刷新

**页面不停刷新？**
→ 点面板里「✖ 取消自动刷新」按钮

**抢完了怎么抢别的课？**
→ 按钮变成「🔄 继续抢课」→ 点它 → 输入新课名 → 再点开抢
