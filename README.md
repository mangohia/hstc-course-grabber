# 🎯 韩师抢课助手

**韩山师范学院选修课自动抢课脚本** — 支持手动输入课程名、拖拽面板、自动点击选课。

## 📥 安装方法

### 前提：安装 Tampermonkey

| 浏览器 | 安装地址 |
|:-------|:---------|
| Chrome | https://www.tampermonkey.net/index.php?browser=chrome |
| Edge   | https://www.tampermonkey.net/index.php?browser=edge |

### 安装脚本

👉 **[点此安装韩师抢课助手](https://gitee.com/mangohia/hstc-course-grabber/raw/main/%E9%9F%A9%E5%B8%88%E6%8A%A2%E8%AF%BE%E5%8A%A9%E6%89%8B.user.js)**

浏览器打开上面链接，Tampermonkey 会自动弹出安装页面，点「安装」即可。

> ⚠️ 如果安装后不生效：浏览器右上角拼图图标 → 管理扩展程序 → Tampermonkey → 详情 → 打开「允许访问文件网址」

## 🚀 使用方法

1. 登录【韩山师范学院教务系统】https://webvpn.hstc.edu.cn
2. 进入 **学生选课** 页面
3. 右上角会出现蓝色面板 **「🎯 韩师抢课助手」**
4. 在文本框中输入要抢的课程名称（每行一个）
5. 点击 **「🚀 立即开抢」** 按钮
6. 脚本自动扫描表格，找到课程并点击选课
7. 全部抢完后显示 **「✅ 全部完成！」**

## ✨ 功能特性

- ✅ **课程名可自由输入** — 想抢什么课自己打字，不用改代码
- ✅ **自动点击选课** — 找到课程后自动点「选课」按钮
- ✅ **每秒重试** — 没刷出来就一直等，持续10分钟
- ✅ **自动处理确认弹窗** — 不用手动点确定
- ✅ **面板可拖动** — 按住标题栏拖到任意位置
- ✅ **随时停止** — 点「⏹ 停止」终止抢课
- ✅ **抢完自动确认** — 切换到「已选课程」标签查看结果

## 💡 小贴士

**提前看好要选什么课！** 最好把课程名字记在备忘录里，抢课的时候直接复制粘贴进去，省得临时翻找手忙脚乱 🎯

## 🔧 自定义配置

```javascript
const AUTO_CHECK = true;     // 抢完后自动切换到「已选课程」标签
const CONFIRM_WAIT = 1500;   // 等待弹窗确认的时间(毫秒)
```

## 📝 注意事项

- 建议提前 **10分钟** 登录教务系统
- 先进入选课页面后再开抢
- 如果面板没出现，检查 🐒 Tampermonkey 图标是否开启
