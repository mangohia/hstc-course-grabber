# 🎯 韩师抢课助手 v5.8

**韩山师范学院选修课自动抢课脚本** —— 每次只抢一门课，抢到停，抢不到继续试。

---

## ✨ 能干什么

| 功能 | 说明 |
|:----|:-----|
| ✅ **输入课程名就能搜** | 打关键词就行，比如「潮州菜」能找到「舌尖上的潮州菜」 |
| ✅ **自动翻遍所有页** | 当前页没找到，自动翻下一页 |
| ✅ **自动点击「选课」** | 找到课程后自动点，不用动手 |
| ✅ **自动处理结果弹窗** | 人数已满 → 自动关窗继续试；抢到 → 自动停 |
| ✅ **轮询检测弹窗** | 200ms 扫一次弹窗，比固定等待快得多 |
| ✅ **页面刷新保活** | 自动模式会定期刷新，防止登录掉线 |
| ✅ **到点自动开抢** | 设好时间，到点自动启动，不用守着 |
| ✅ **无限重试** | 不设次数上限，直到抢到或手动停止 |
| ✅ **面板可拖动** | 按住标题栏拖到不挡视线的地方 |
| ✅ **多浏览器** | Chrome / Edge / 360 都能用 |

---

## 🚀 30 秒学会

```
装好脚本 → 打开教务系统选课页面
→ 右上角出现蓝色面板
→ 输入课程名（每次只输一门）
→ 点「⚡ 立即开抢」
→ 等着，抢到了自动停
→ 点「🔄 继续抢课」换下一门再抢
```

## 📝 使用技巧

**⏰ 提前登录**
提前 10 分钟打开教务系统并登录，别卡在登录那步

**📋 提前存好课程名**
把要抢的课程名存在备忘录里，抢课时从高到低依次输入

**🎯 先抢最想要的**
抢到一门 → 继续抢下一门 → 全抢完收工

**🤖 用自动模式省心**
还没到选课时间，用自动模式保活 + 到点自动开抢：

```
① 输入课程名
② 设时间（如 14:00）
③ 勾选「自动刷新保活」
④ 点「🚀 启动自动模式」
→ 页面自动刷新保活
→ 到 14:00 自动开抢
→ 你到点回来看结果就行
```

**💡 课程名不用打全**
输入关键词就行，「潮州菜」也能匹配到「舌尖上的潮州菜」

## 📥 安装方法

### 1️⃣ 装 Tampermonkey（🐒 猴子插件）

| 浏览器 | 安装地址 |
|:-------|:---------|
| **Chrome** | https://www.tampermonkey.net/index.php?browser=chrome |
| **Edge** | https://www.tampermonkey.net/index.php?browser=edge |
| **360/QQ 浏览器** | 跟 Chrome 一样，用上面的链接 |

点链接 → 点「Download」或「添加至 Chrome」→ 添加扩展程序

> **安装成功：** 浏览器右上角出现 🐒 图标

> ⚠️ **安装后必做这一步！**
> ① 点右上角 🧩 **拼图图标** → **管理扩展程序**
> ② 找到 **Tampermonkey** → 点 **「详情」**
> ③ 往下翻 → 打开 **「允许访问文件网址」** 开关
> 
> ❌ 不打开这个开关，脚本不会运行！

### 2️⃣ 装抢课脚本

👉 **[点此一键安装](https://gitee.com/mangohia/hstc-course-grabber/raw/main/%E9%9F%A9%E5%B8%88%E6%8A%A2%E8%AF%BE%E5%8A%A9%E6%89%8B.user.js)**

点上面的链接 → 浏览器弹出安装页面 → 点 **「安装」** → 完成！

> **手动安装（一键安装没反应时）：**
> ① 右键上面的链接 → 「另存为」→ 保存到桌面
> ② 点 🐒 图标 → **「管理面板」**
> ③ 点 **「➕ 添加新脚本」**
> ④ 删掉默认代码
> ⑤ 用记事本打开刚才下载的文件 → 全选复制（Ctrl+A → Ctrl+C）
> ⑥ 粘贴到浏览器（Ctrl+V）
> ⑦ **Ctrl+S** 保存

### 3️⃣ 开抢！

打开教务系统 → 登录 → 进入选课页面（URL 含 `eams/stdElectCourse`）
→ 右上角出现蓝色面板 → 输入课程名 → 开抢！🎯

## ❓ 常见问题

**面板没弹出来？**
```
① 检查 🐒 是否灰色 → 点一下变彩色
② 检查 URL 是否为选课页面（含 eams）
③ 检查「允许访问文件网址」是否打开
④ 刷新页面
⑤ 重装脚本
```

**显示「未出现」？**
→ 课程还没显示在这个页面，脚本会自动翻页找，等一等

**页面不停刷新？**
→ 点面板里的「✖ 取消自动刷新」按钮

**抢完了怎么抢别的课？**
→ 按钮变成「🔄 继续抢课」→ 点它 → 输入新课名 → 再点开抢

**提示「人数已满」一直抢不到？**
→ 脚本会无限重试，有位置空出来就会抢到。如果确定没希望了，点「⏹ 停止」
