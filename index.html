📥 安装方法

1️⃣ 装 Tampermonkey

浏览器安装地址
Chrometampermonkey.net
Edgetampermonkey.net
360 / QQ 浏览器跟 Chrome 一样，用上面的链接
⚠️ 安装后必做这一步！
① 点右上角 🧩 拼图图标 → 管理扩展程序
② 找到 Tampermonkey → 点「详情」
③ 往下翻 → 打开「允许访问文件网址」开关
❌ 不打开，脚本不会运行！

2️⃣ 装抢课脚本

📦 一键安装

点上面的链接 → 浏览器弹出安装页面 → 点「安装」→ 完成！

手动安装教程（一键安装没反应时）
① 右键上方「一键安装」→ 另存为 → 保存到桌面
② 点 🐒 图标 → 管理面板
③ 点「➕ 添加新脚本」
④ 删掉默认代码
⑤ 用记事本打开下载的文件 → Ctrl+A → Ctrl+C
⑥ 粘贴到浏览器 → Ctrl+V
⑦ Ctrl+S 保存

3️⃣ 开抢！

→ 打开教务系统 → 登录
→ 进入选课页面（URL 含 eams/stdElectCourse）
→ 右上角出现 蓝色面板
→ 输入课程名 → 点「⚡ 立即开抢」🎯

📝 使用技巧

⏰ 提前登录
提前 10 分钟打开教务系统并登录，别卡在登录那步
📋 提前存好课程名
把想抢的课存在备忘录里，从高到低依次输入，一门一门抢
🤖 自动模式最省心
还没到选课时间？
① 输入课程名
② 设时间（如 14:00）
③ 勾选「自动刷新保活」
④ 点「🚀 启动自动模式」
页面自己刷新保活 → 到点自动开抢 → 你回来看结果
💡 课程名不用打全
输入关键词就行，「潮州菜」也能匹配到「舌尖上的潮州菜」

❓ 常见问题

面板没弹出来？
① 点右上角 🧩 拼图图标 → 管理扩展程序
② 找到 Tampermonkey（篡改猴）→ 点「详情」
③ 打开「允许访问文件网址」开关
④ 在 Tampermonkey 管理面板中，确认脚本已启用
⑤ 检查 URL 是否为选课页面（含 eams）
⑥ 刷新页面
显示「未出现」？
课程还没显示在这个页面，脚本会自动翻页查找，等一等就行
页面不停刷新？
点面板里的「✖ 取消自动刷新」按钮
抢完了怎么抢别的课？
按钮变成「🔄 继续抢课」→ 点它 → 输入新课名 → 再点开抢
一直「人数已满」抢不到？
脚本会无限重试，有人退课就会抢到。如果确定没希望了，点「⏹ 停止」
